NeurIPS 2026 审稿报告：ADAMA
论文标题：ADAMA: Entropy-Guided Operator Fusion with Hadamard Reparameterization for Efficient State Space Models
总体评分（模拟）：4 / 10 — Borderline Reject
置信度：4 / 5

一、总结
本文提出 ADAMA，一个将激活熵作为运行时信号，结合 Hadamard 重参数化，动态决定 SSM 算子融合边界与分块粒度的框架。核心思想是：Hadamard 旋转使激活分布更均匀（熵升高），从而允许更深的融合而不引起数值不稳定，这一机制同时服务于量化鲁棒性和内存效率。理论部分包含三个命题（熵不减、融合深度上界、量化稳定性），并给出完整证明。
问题在于，所有实验结论均来自一个合成数据的 NumPy/PyTorch 原型——无真实 GPU 内核，无 Triton 实现，无在实际预训练模型上的系统级 benchmark。唯一的真实模型实验是在 CPU 上 FP32 运行 Mamba-1.4B，并坦承该结果"不具代表性"。这使得论文的核心系统贡献主张缺乏实证支撑，当前状态不适合在系统类顶会发表。

二、优点

问题定义清晰：将 SSM 内存瓶颈、激活异常值与算子融合三者统一在一个框架内，思路连贯。
理论部分完整：Theorem 1 的证明（Jensen 不等式 + 双随机矩阵）是数学上严格的，附录中有完整展开。
写作结构良好：各节逻辑清晰，附录内容充实，可复现性清单详尽。
适应性阈值设计合理：自适应阈值 τ_t 和 DP 优化器的设计具有实践价值。


三、主要缺陷（Major Weaknesses）
W1：实验基础不成立（致命问题）
所有系统性能数据来自合成激活上的 Python 原型，延迟/吞吐量等数值实际上是模拟值，而非真实 GPU 内核执行时间。论文在 Section 5.1 自述：

"Our evaluation is a controlled prototype study rather than a full end-to-end Triton benchmark."

NeurIPS 系统方向论文要求在真实硬件上基于实际内核的 benchmark。没有 Triton/CUDA 内核，"25% 延迟降低"等数字无法被验证或复现。
建议：在提交前完成至少一个精度设置（FP16）下的真实 Triton 融合内核实现，并在 GPU（如 A100/RTX 3070）上测量实际延迟。
W2：与 baseline 的比较缺乏公平性
"Static Fusion"和"No Fusion"均为同一原型系统内的模拟选项，不是独立实现的竞争系统。论文未与以下工作比较：

FlashMamba / Mamba2 的官方融合实现
GPTQ/AWQ 等量化基线的实际推理性能
针对 SSM 的既有 Triton 内核（如 mamba-ssm 官方库）

latex\subsection{Entropy Growth Under Rotation}

\begin{theorem}[Entropy Increase Under Doubly-Stochastic Histogram Mixing]
\label{thm:entropy}
Let $p \in \Delta^m$ be the empirical histogram mass vector of activations
before rotation, and let $q \in \Delta^m$ be the histogram mass vector after
rotation using the same fixed bins.  If there exists a doubly-stochastic
matrix $B \in \mathbb{R}^{m \times m}$ such that $q = Bp$, then the
finite-bin entropy estimate satisfies $H_{\mathrm{hist}}(q) \ge
H_{\mathrm{hist}}(p)$.  If $B$ is not a permutation and $p$ is non-uniform,
the inequality is strict.
\end{theorem}

\noindent\textit{Proof sketch.}
The function $\phi(t) = -t \log t$ is strictly concave on $[0,1]$.
Applying Jensen's inequality to each coordinate
$q_i = \sum_j B_{ij} p_j$ yields
$\phi\!\left(\sum_j B_{ij} p_j\right) \ge \sum_j B_{ij}\phi(p_j)$.
Summing over $i$ and using the column-sum property of a doubly-stochastic
matrix gives $H_{\mathrm{hist}}(q) \ge H_{\mathrm{hist}}(p)$.
Strictness follows when at least one row of $B$ mixes unequal masses.
See Appendix~\ref{sec:appendix_proofs} for the full proof.

\noindent\textbf{Applicability condition.}
Theorem~\ref{thm:entropy} requires the post-rotation histogram to be
representable as a doubly-stochastic mixing of the pre-rotation histogram
over the chosen fixed bins.  We verify this condition empirically in
Appendix~\ref{sec:appendix_proofs}: for all tested SSM layers and sequence
lengths, the measured Birkhoff decomposition error is below $1\%$ of the
total histogram mass.

\begin{theorem}[Shared-Memory Depth Bound]
\label{thm:depth}
Let $M_{\mathrm{shared}}$ denote the available shared memory per thread block,
and let $C_{\mathrm{tile}}$ denote the incremental memory footprint
contributed by each additional fused operator under a fixed tile shape.
Then the fusion depth $F$ satisfies
\[
  F \;\le\; \frac{M_{\mathrm{shared}}}{C_{\mathrm{tile}}}.
\]
\end{theorem}

\noindent\textit{Proof sketch.}
Each additional fused operator contributes a nonneg\-ative memory cost to the
on-chip working set.  Feasibility requires $F C_{\mathrm{tile}} \le
M_{\mathrm{shared}}$; rearranging gives the bound.
The full proof is in Appendix~\ref{sec:appendix_proofs}.

latex\subsection{Stability Under Quantization}

\begin{theorem}[Variance Preservation with Reduced Coordinate Extremes]
\label{thm:quant}
Let additive quantization noise be modeled as $\epsilon \sim \mathcal{N}(0,\sigma^2)$.
After orthonormal Hadamard rotation, total variance is preserved,
$\mathrm{Var}(Hx) = \mathrm{Var}(x)$,
but the tail probability of a single coordinate exceeding a fixed clipping
threshold $k\sigma$ is reduced for heavy-tailed activations:
\[
  P\!\left(|(Hx)_i| > k\sigma\right) < P\!\left(|x_i| > k\sigma\right).
\]
\end{theorem}

\noindent\textit{Proof sketch.}
Orthogonal transforms preserve second moments globally but spread large
coordinates over many dimensions, lowering the chance that any one channel
dominates the dynamic range.
See Appendix~\ref{sec:appendix_proofs} for the complete argument.

  \Delta Q = \frac{\widehat{H}_{\text{fused}}(Z) - \widehat{H}_{\text{ref}}(Z)}

七、结论
ADAMA 提出了一个概念上有说服力的框架，将 Hadamard 平滑与熵驱动的融合调度统一起来，理论分析也较为严谨。然而，缺乏真实 GPU 内核实验是当前版本的根本性缺陷：所有系统性能数字来自 Python 原型，无法支撑"ADAMA 在实际 GPU 上优于现有融合方案"的主张。建议作者在后续版本中：

完成 Triton FP16 融合内核并在真实 GPU 上测量；
与 mamba-ssm 官方实现和 FlashAttention 风格的 baseline 做公平比较；
在实际预训练 Mamba 模型上报告 perplexity 和 LongBench 分数。

满足以上三点后，这篇论文有潜力成为 NeurIPS 系统赛道的高质量投稿。
